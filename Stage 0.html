Key Ideas

The Internet

The Internet is the world's largest computer network, and it uses HTTP as its main protocol. Through the Internet, individuals can access files stored on servers through a browser. These servers are essentially computers that are optimised for storing files, and they can store any type of file to be made accessible over the internet. Most of the content that can be found on the internet consists of HTML (Hypertext Markup Language).

HTML

HTML documents can consist of the following:

  • Text content: This is what you see
  • Markup: This indicates what it looks like
  • References to other documents: This can be images or videos embedded in the page; it can also be references to CSS style sheets that indicates how the page looks like
  • Links to other pages

Markup usually consists of an opening and a closing tag (although there are exceptions). An element referrs to an open and close tag plus the content between them.

Elements can be either inline or block. Block elements automatically create a kind of "invisible box" around them, whereas inline elements don't. This means that, unless explicitly stated, inline elements will be displayed on one continues line.
